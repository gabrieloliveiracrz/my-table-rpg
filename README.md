# My Table RPG

This is a personal project that I am building to learn software architecture, game development, and full-stack engineering.

My goal is to create my own virtual tabletop RPG platform so I can play with my friends.

The idea is to combine the campaign management experience of Roll20 with the tactical turn-based gameplay of games like Baldur's Gate 3 and Divinity: Original Sin 2.

## Why I'm Building This

I wanted a more modern and interactive RPG experience than traditional virtual tabletops.

Instead of using static tokens and relying entirely on imagination, characters will become fully interactive entities with:

- Actions
- Abilities
- Inventory
- Turn-based combat
- Tactical movement

At the same time, the platform will provide tools for game masters to:

- Create campaigns
- Schedule sessions
- Upload maps and music
- Manage monsters and encounters

## Project Goals

- Learn software architecture and design patterns
- Practice full-stack development
- Explore game development on the web
- Build a real product for my own RPG sessions
- Create a strong portfolio project
